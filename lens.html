The problem

Most developers review code without enough context or control.

Cloud AI tools send your code off-machine. Chat sessions forget previous decisions. Switching models is painful. And structured, repeatable reviews are rare.

What Lens does

Local. Multi-model. Persistent. Built for real reviews.

Lens is a lightweight Python agent that runs entirely on your machine. It uses LiteLLM so you can switch between Claude, GPT, Gemini, Grok or local models with one environment variable. It keeps persistent memory across sessions and comes with purpose-built workflows for architecture, security, performance and maintainability reviews.
